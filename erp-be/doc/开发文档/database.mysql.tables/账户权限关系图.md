# 账户，角色，权限 表字段与关系图

## 表格

### sys_account

```mysql
CREATE TABLE `sys_account` (
    `id` bigint NOT NULL,
    `created_by` bigint NOT NULL,
    `created_date` datetime NOT NULL,
    `updated_by` bigint NOT NULL,
    `updated_date` datetime NOT NULL,
    `deleted` tinyint(1) NOT NULL,
    `username` varchar(45) CHARACTER SET utf8mb4 COLLATE utf8mb4_bin NOT NULL,
    `password` varchar(255) CHARACTER SET utf8mb4 COLLATE utf8mb4_bin NOT NULL,
    `type` varchar(10) COLLATE utf8mb4_bin NOT NULL,
    `enabled` tinyint(1) UNSIGNED ZEROFILL NOT NULL,
    PRIMARY KEY (`id`),
    KEY `I_QUERY` (`deleted`, `username`, `password`, `enabled`, `type`) USING BTREE
) ENGINE = InnoDB
  DEFAULT CHARSET = utf8mb4
  COLLATE = utf8mb4_bin
```

### sys_role

```mysql
CREATE TABLE `sys_role` (
    `id` bigint NOT NULL,
    `created_by` bigint NOT NULL,
    `created_date` datetime NOT NULL,
    `updated_by` bigint NOT NULL,
    `updated_date` datetime NOT NULL,
    `deleted` tinyint(1) NOT NULL,
    `role_name` varchar(255) CHARACTER SET utf8mb4 COLLATE utf8mb4_bin NOT NULL,
    `parent_id` bigint DEFAULT NULL,
    `enabled` tinyint NOT NULL,
    PRIMARY KEY (`id`),
    KEY `I_QUERY` (`deleted`, `role_name`, `enabled`) USING BTREE
) ENGINE = InnoDB
  DEFAULT CHARSET = utf8mb4
  COLLATE = utf8mb4_bin
```

### sys_role_account_association

```mysql
CREATE TABLE `sys_role_account_association` (
    `id` bigint NOT NULL,
    `created_date` datetime NOT NULL,
    `created_by` bigint NOT NULL,
    `updated_date` datetime NOT NULL,
    `updated_by` bigint NOT NULL,
    `deleted` tinyint(1) DEFAULT NULL,
    `account_id` bigint NOT NULL,
    `role_id` bigint NOT NULL,
    `enabled` tinyint(1) UNSIGNED ZEROFILL NOT NULL,
    PRIMARY KEY (`id`),
    KEY `I_QUERY` (`deleted`, `enabled`, `account_id`, `role_id`) USING BTREE
) ENGINE = InnoDB
  DEFAULT CHARSET = utf8mb4
  COLLATE = utf8mb4_bin
```

### sys_permission

```mysql
CREATE TABLE `sys_permission` (
    `id` bigint NOT NULL,
    `created_by` bigint NOT NULL,
    `created_date` datetime NOT NULL,
    `updated_by` bigint NOT NULL,
    `updated_date` datetime NOT NULL,
    `deleted` tinyint(1) NOT NULL,
    `business_group` varchar(50) COLLATE utf8mb4_bin NOT NULL,
    `permission_name` varchar(50) CHARACTER SET utf8mb4 COLLATE utf8mb4_bin NOT NULL,
    `type` varchar(10) COLLATE utf8mb4_bin NOT NULL,
    `access_resource` varchar(200) CHARACTER SET utf8mb4 COLLATE utf8mb4_bin NOT NULL,
    `enabled` tinyint(1) UNSIGNED ZEROFILL NOT NULL,
    PRIMARY KEY (`id`),
    KEY `I_QUERY` (`deleted`, `business_group`, `permission_name`, `type`, `access_resource`, `enabled`) USING BTREE

  DEFAULT CHARSET = utf8mb4
  COLLATE = utf8mb4_bin
```

## 关系

- sys_account.id -> sys_role_account_association.account_id

- sys_role.id -> sys_role_account_association.role_id

- sys_role.id -> sys_permission_role_association.role_id

- sys_permission.id -> sys_permission_role_association.permission_id
